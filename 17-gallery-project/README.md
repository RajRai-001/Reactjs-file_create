# 📸 Photo Gallery

A modern and responsive **Photo Gallery Web App** built with **React**, **Tailwind CSS**, and **Axios**. The application fetches beautiful images from the Picsum API and provides a clean UI with search, pagination, and download features.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/RajRai-001/Reactjs-file_create/tree/main/17-gallery-project
```

Go to project folder

```bash
cd photo-gallery
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
